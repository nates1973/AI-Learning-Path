# Enterprise AI Architect Learning Path

These documents describe a learning path for achieving expert level skills in agentic AI systems implementation and 
architecture. It presents a structured, progressive set of study and exercises to ensure learners gain the critical 
knowledge and hands-on experience in delivering enterprise AI solutions.

## How to approach the learning path

**Overview**
The overview document presents the backbone of the learning path. It outlines, at a high level, all the major phases 
the learner will navigate in gaining these skills. Each of the major phases is then detailed in its own Phase Guide document.

**Phase Guides**
Each Phase Guide details the goals and tasks for that phase, along with clear definitions of done and standards for success.

**Resource Catalog**
The resource catalog provides a notated list of all the learning materials needed to complete the work in the various phases 
of the learning path.

**Planning Prompt**
The planning prompt is intended for the AI coding assistant you will be using to help complete this work. The assistant 
(GitHub Copilot, Claude Code, etc.) will be operating as both scrum master and mentor to guide the learner through the various 
phases. Based around one-week sprints, this prompt will help the chosen coding assistant to plan the week's work based on all
previous progress.

## Basic process
As noted, the learner is expected to work with an AI coding assistant (GitHub Copilot inside Visual Studio Code is recommended), 
to complete the various phases. Further, it is assumed that the learner has access to the following resources:
- Udemy Enterprise
- O'Reilly Learning Platform
- Microsot Learn
- Microsoft Azure
- Azure Foundry
- GitHub

As a learner, you should first familiarize yourself with the Overview document so you have a clear picture of the learning process
you are beginning. It will be your discretion as to whether you need more or less work in any given area based on your current 
experience. For example, if you already have a solid foundation in Python programming, you may not require the complete 100 Days
of Code Python course in Phase 1.

Once you are comfortable with the overall learning path, begin your work by studying the Phase 1 Guide. When you are comfortable 
with the contents of Phase 1, and have set up all the necessary tools, call up your coding assistant, provide the Overview, 
Resource Catalog and Phase Guide 1 as context resources, and use the Prompt to generate your first week's sprint plan.

Once your sprint plan is ready, get going! Good luck!
